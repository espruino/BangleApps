# banglejs-countdown
Simple countdown app that displays a live countdown to a chosen date which is stored.

## Screenshots
![Countdown screenshot 1](countdown-screenshot-1.png)
![Countdown screenshot 2](countdown-screenshot-2.png)

## Usage
- Push the button shortly to select the goal date.

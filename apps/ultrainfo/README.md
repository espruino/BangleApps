# Ultra Info Clock
For those who crave a clock with information, this clock has 6 clockInfos. Uses `ClockBG` module for enhanced clock backgrounds.

Additionally, each clockInfo has a bar at the bottom, and if the clockInfo on display has a `max` value, it displays a progress meter there. (Steps, heart rate, altitude, etc.)

## Settings:
- `Accent Color` - The color of the progress bars and ClockInfos when focused.

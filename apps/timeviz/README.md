# Time Viz

A visual timer app for Bangle.js 2 that helps with time blindness, designed especially for people with ADHD.

## Features

- **Radial countdown visualization**: Watch time pass with an intuitive circular display
- **High contrast design**: Red countdown area for maximum visibility
- **Configurable duration**: Set timers from 1 to 60 minutes
- **Simple controls**: Tap to start, pause, or reset
- **Vibration alert**: Get notified when your timer completes
- **Always visible**: Clear time display in the center

## How to Use

1. **Configure the timer**: Use the app settings in the App Loader to set your preferred duration (default is 25 minutes)
2. **Launch the app**: Open Time Viz from your watch
3. **Start the timer**: Tap anywhere on the screen to start the countdown
4. **Monitor progress**: Watch the red area decrease as time passes
5. **Pause if needed**: Tap to pause, tap again to resume
6. **Complete**: When time runs out, the watch will vibrate and you can tap to reset

## Visual Design

The app features a radial timer where:
- A red-filled circle shows the remaining time
- The filled area decreases clockwise as time passes
- The center displays the exact time remaining (MM:SS)
- Status text at the bottom shows the current state (Running, Paused, Ready, etc.)

## Settings

- **Timer Duration**: Configure from 1 to 60 minutes
  - Default: 25 minutes (Pomodoro technique)
  - Access via App Loader settings page

## Accessibility

This app is designed to help people with time blindness:
- **Visual feedback**: See time passing in real-time
- **High contrast**: Red on black for clear visibility
- **Simple interface**: No complex menus or settings
- **Immediate feedback**: Know at a glance how much time remains

## Tips

- Use the default 25-minute setting for the Pomodoro technique
- Set shorter durations (5-10 min) for quick tasks
- Use longer durations (45-60 min) for focused work sessions
- The button on the watch returns to the launcher

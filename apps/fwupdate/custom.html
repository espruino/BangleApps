Firmware updates via this tool work differently to the NRF Connect method mentioned on the Bangle.js 2 page. Firmware is uploaded to a file on the Bangle. Once complete the Bangle reboots and the bootloader copies the new firmware into internal Storage.

In addition to the links above, you can upload a hex or zip file directly below. This file should be an .app_hex file, *not* the normal .hex (as that contains the bootloader as well).

DANGER! No verification is performed on uploaded ZIP or HEX files - you could potentially overwrite your bootloader with the wrong binary and brick your Bangle.

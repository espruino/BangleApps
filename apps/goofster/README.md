# Goofster

## What is a Goofster

Goofster is a time telling friend, his left eye tells the hour while his right eye tells the minutes. Scientists are still working on if he displays seconds in any way. 

Goofster's colour is customisable in the settings menu, as well as the widget bar (which I like to think gives him a little hat)


## How did he get here

Goofster is a little watchface I made for Pebble watches for their spring contest in 2026. It can be found here on the [app store](https://apps.repebble.com/goofster_56599303db6c403ea523cb8a) or on [github](https://github.com/LochieDoesStuff/Goofster)

Unfortunately, I don't currently daily drive a pebble, they're just kinda relics that still work, as long as you have a charger around. So instead, I ported it across here to Bangle! Which I do wear in my rotation. It's mostly the same! Which I think is pretty neat considering the differences in capability of the watches! Drawing arcs was a real pain, if anyone has a better way to draw an arc with stroke width, I would love to hear it.

This is my first real JS project for me(I think? Definitely the first public one and one I could say is actually DONE) and I think it goes okay! Hasn't blown up my watch yet! I could definitely make some improvements, clean up the code a little, but honestly I am happy with how it is now.

Shoutout to [David Peer](https://github.com/peerdavid), who's settings code from LCARS I 100% cribbed, the colours there were set out great and they look good on the display.

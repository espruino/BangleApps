
Intentional
========

Store task names and intents in a json file - intentional.json
Show tasknames in a menu on the watch - intentional.app.js
Allow users to load the contents of intentional.json from the browser and add/remove/edit/sort the tasks - intentional.html


Set-Up
--------
- `Task Name` - The name of the task
- `Intent` - Say something about that
- `GadgetBridge` - say something about enabling that
- `Tasker` - say something about setting up Tasker profiles and tasks

# Dotmatrix clock

A clock face simulating the classic dotmatrix displays. Shows time, date, compass, and heart rate.

![](dotmatrix-clock-screen-shot.png)

## Features

* Easy to read digits
* Simulated white-on-blue dotmatrix display
* Compass
* Heart rate monitor
